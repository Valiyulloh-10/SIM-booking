# Project: Next.js + Express + React Admin

## How to run locally

1. Install dependencies in each folder:
```bash
cd api && npm install
cd ../web && npm install
cd ../admin && npm install
```

2. Start backend:
```bash
cd api && npm run dev
```

3. Start frontend:
```bash
cd web && npm run dev
```

4. Start admin:
```bash
cd admin && npm run dev
```

Or use docker-compose:
```bash
docker-compose up --build
```
